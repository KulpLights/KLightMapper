# KLightMapper (binary releases)

Prebuilt `KLightMapper.xcframework` for the xLights camera-based
light-mapping feature. Source is maintained privately; this repo
publishes only the binary framework for Swift Package Manager.

```swift
// pinned stable
.package(url: "https://github.com/KulpLights/KLightMapper.git", from: "1.0.0")
// rolling pre-release
.package(url: "https://github.com/KulpLights/KLightMapper.git", branch: "latest")
```
